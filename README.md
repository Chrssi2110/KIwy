## Kurzer Hinweis zum Testen 

Zum lokalen Testen gibt es zwei Optionen:

- Direkt im Browser öffnen:

	- Datei `index.html` im Projektordner doppelklicken oder per Browser öffnen.

- Lokaler HTTP-Server (empfohlen):

	- Projekt in einer IDE (VS Code, Antigravity, Cursor, o.Ä. öffnen)
    - Mit Python 3 (im Terminal im Projektordner ausführen):

python3 -m http.server 8000

- Danach im Browser öffnen: http://localhost:8000

--- 

- Alternative mit Node (falls Node installiert):
npx http-server -p 8000

- Danach im Browser öffnen: http://localhost:8000
